# Comparativo entre Python e JavaScript

## Operadores lógicos

**Python**
```python
if a and b:
    # executa se ambos forem verdadeiros
if a or b:
    # executa se algum for verdadeiro
if not a:
    # executa se “a” for falso
```

**JavaScript**

```Javascript
if (a && b) {
    // executa se ambos forem verdadeiros
}
if (a || b) {
    // executa se algum for verdadeiro
}
if (!a) {
    // executa se “a” for falso
}
```
